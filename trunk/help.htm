About:

Fast Transit is a project that I have been working on in my spare time to speed up the look up time for transit bus times. The idea was to produce an interface with usability and speed of use in mind.
   -- Andy Lumb - Lynet.ca

How To:

  • How To Find bus times
    1. Fill in route name - Hit <enter> to select highlighted route
    2. Fill in any portion of the stop name - Hit <enter> to select highlighted stop
    3. Times are displayed in frame - Next bus is highlighted
  • Mini Print
    1. Select your stops using the normal interface
    2. Click "miniprint" and a new window will open
    3. Print
    4. Fold between tables
    5. Cut down to 3"x5"
    6. Recommended: cover in plastic book cover sheets
